WHAT IS A NETWORK?

A computer network is a group of computers or devices connected together so they can communicate and exchange data with each other.

Examples of networks we encounter every day include telephone networks, satellite networks, computer networks, and the Internet.

This module gives you the big picture of networking — understanding what devices exist in a network and what they are called, before diving into how each device actually works.

🚗 Think of it like a Car

Before understanding how a transmission works, you first need to know a car has an engine, wheels, and gears. Networking is the same — learn the parts first, then learn how they work.

NETWORK DEVICES

Every network is built from these core hardware components. Understanding each device's role is your first step toward mastering networking.

Router

Connects one network to another network. Routers handle the complex task of directing data packets between different networks — including your home network to the Internet. More sophisticated than a Switch.

Layer 3 — Network Layer
Switch

The central hub of a wired network. Connects multiple computers and network devices together. Unlike a Hub, a Switch is intelligent — it sends data ONLY to the intended destination port, not broadcast to all ports.

Layer 2 — Data Link Layer
Wireless Access Point

Works like a Hub but wirelessly. Allows Wi-Fi devices (laptops, phones) to connect to the network without cables. Acts as a bridge between wireless clients and the wired network infrastructure.

Wireless — 802.11
Firewall

The network's security guard. Placed between the external network (Internet) and internal network (Intranet), it monitors and controls incoming and outgoing traffic. Can be hardware or software-based.

Security — Perimeter Defense
Server

A powerful computer that provides services to other computers (clients). Common server types include Web servers, Email servers, DNS servers, and Database servers. Requires high-performance hardware to handle many simultaneous requests.

NETWORK SIZES

Networks come in three main sizes. Each has different requirements, equipment, and complexity levels.

Home / Small Network

Found in homes, dormitories, and condominiums. Internet access is provided by an ISP (Internet Service Provider) such as TOT, TRUE, 3BB, or AIS.

Typically uses ADSL (copper wire connection) or FTTH (Fiber to the Home via optical fiber cable). The ISP provides either an ADSL Router or an ONU/ONT Optical Modem.

Home networks usually use all-in-one devices that combine a Router, Switch, and Access Point into one unit.

Wi-Fi Modem/Router ADSL / Fiber 2–10 Devices Home ISP Package
  • All-in-one modem/router/AP device from ISP
  • Simple setup — plug and play
  • Shared bandwidth among all home devices
  • Basic security (home-grade firewall)

Small / Medium Business

Networks for SMEs are more complex than home setups. Devices are no longer all-in-one — Router, Switch, and Access Point are separate dedicated hardware units.

Internet connectivity uses a Business Package designed for SME-level reliability, with better speeds and SLA guarantees compared to home packages.

Dedicated Router Managed Switch Access Points Business ISP 10–200 Devices
  • Separate dedicated hardware for each role
  • Managed switches with VLAN support
  • Dedicated firewall appliance
  • Business-grade SLA from ISP

Enterprise / Large Organization

Enterprise networks demand the highest level of reliability. Internet connectivity must have redundancy and High Availability — if any device fails, another takes over seamlessly with no downtime.

There should be no single point of failure anywhere in the system. For large, complex networks with many devices, SDN (Software Defined Networking) helps with Network Automation and management.

Core / Dist / Access Redundant Links High Availability SDN / Automation 1000+ Devices
  • 3-tier architecture: Core, Distribution, Access
  • Redundant connections — no single point of failure
  • Enterprise firewall with IPS/IDS
  • SDN for network automation and management
  • Dedicated NOC (Network Operations Center)
